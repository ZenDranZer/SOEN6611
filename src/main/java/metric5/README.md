SciTools Understand:

Register and download a trial version of the SciTools Understand by going to the website.

Click on the downloaded .exe file and install Understand.

Click on Continue evaluation.

Create a new project File -> New -> Project or (Alt + F + N + P)

Specify the project directory and click on create project.

To generate Knots report, Click on Metrics -> Export metrics or (Alt + M + E).

Select Knots checkbox -> Export It will export a .csv file to the specified destination.
